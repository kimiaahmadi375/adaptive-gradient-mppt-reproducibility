# Code

This folder contains the custom implementation of the proposed adaptive gradient-descent-based MPPT algorithm.

## Files

- `MAin_RTL_Based_My_Algorithm_2026_05_12.txt`: Text representation of the proposed MPPT algorithm, including the adaptive gradient-based duty-cycle update, threshold-based perturbation suppression, and optional initialization routine for partial-shading conditions.

## Notes

The algorithm is intended to document the custom computational procedure used in the study. The implementation parameters and supporting simulation data are provided in the `simulation_data` folder.
